# Air Paint

**Air Paint** allows you to draw on a window by moving a **red**-colored object through the air, which is tracked via your webcam.

## Languages / Tools Used
- Python 3.8 (will update version support in the future)

## How to Run
1. Ensure the `numpy`, `cv2`, and `PIL` modules are installed
2. Run `Air Paint.py`
